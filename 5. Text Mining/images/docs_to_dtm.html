Creation of semantic variables based on the document-term matrix (DTM)

SPE3DLab allows you to add the variables derived from the DTM analysis of the selected variables.

The DTM analysis can either choose the concatenation of the selected variables or on each of the variables.
The following pretreatments are automatically applied: deletion of stop words, wordstemming. These pre-treatments relate to the language specified by the user.
